Cover Finder

Cover Finder is an Open Source Simple Application written with Angular 4 which Helps you to find albums, their Art Covers, details and Listen to previews with the help of Spotify's Web Api.

Source Code is Available on GitHub (Cover Finder App), Feel free to Contribute or Fork and modify it. There's also an API Wrapper written on top of Spotify's API hosted on Heroku which you can find on GitHub (Cover Finder Api) as well.
